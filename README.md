# Aplicativo Interactivo de Lógica y Algoritmos

Este proyecto es una herramienta interactiva desarrollada en C++ para la asignatura de Algoritmos y Lógica de Programación de la Universidad Técnica de Ambato. El sistema integra operaciones matemáticas básicas, procesamiento de notas académicas y almacenamiento de información mediante archivos de texto.

## Características Principales

### Menú Interactivo
Interfaz de consola organizada que permite acceder fácilmente a las diferentes funcionalidades del programa.

### Operaciones Matemáticas
Implementación de:
- Suma
- Resta
- Multiplicación
- División con validación para evitar división por cero.

### Procesamiento de Notas
Registro automático de 5 notas utilizando arreglos y estructuras repetitivas para calcular:
- Promedio
- Nota mayor
- Nota menor
- Cantidad de aprobados
- Cantidad de reprobados

### Persistencia de Datos
Generación automática del archivo resultados.txt donde se almacenan:
- Nombre del estudiante
- Resultados obtenidos
- Fecha
- Lenguaje utilizado

### Robustez
Uso de validaciones y estructuras condicionales para evitar errores durante la ejecución del programa.

# Estructura del Proyecto

/src  
Contiene el código fuente principal del programa desarrollado en C++.

/resultados  
Carpeta donde se almacena el archivo resultados.txt generado automáticamente.

/capturas  
Contiene evidencias visuales y capturas del funcionamiento del programa.

/README.md  
Archivo descriptivo del proyecto y documentación general.

# Instrucciones de Uso

## Preparación
Asegurarse de tener instalado un compilador de C++ o Visual Studio Code con extensiones para C++.

## Compilación
Abrir el archivo main.cpp y compilar el programa utilizando cualquier compilador estándar de C++.

## Ejecución
Ejecutar el programa y seleccionar las opciones del menú interactivo según la funcionalidad deseada.
